大致流程

安装

熟练？使用命令行

安装Hugo:使用winget

winget install Hugo.Hugo.Extended

安装git:Git - 下载软件包 - Git 版本控制系统

安装Go:欸我好像没有用到

验证

hugo version

搭建新站点

选择喜欢的hugo主题

官方主题网站: https://themes.gohugo.io/

Hugo官方的教程提供了详细的命令解释和例程：快速入门 | Hugo 框架

hugo new site 站点名字

cd 站点名字

git init

git submodule add [fork喜欢的主题得到的网址]

然后在根目录下的 config.toml文件中添加新的一行:

theme = "主题名"

这一步使用教程里面的echo方法可能会出现问题，建议二次确认一下

启动 Hugo 的开发服务器以查看站点

hugo server

如果顺利的话，可以在http://localhost:1313/看到本地部署的网站
